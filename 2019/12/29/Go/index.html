Packeages, variables, and functions

Packages

包的名字是输入路径的最后一个元素，例如”math/rand”包的声明是rand，在使用中我们可以这样使用rand.Seed(3)和rand.Intn(10)。

Imports

和

是等价的。

Exported names

如果一个变量用大写字母开头，那么这个变量是一个exported name。例如Pi。

Pi是math package中的常量，可以用math.Pi来表示。

Functions

Functions continued

(x int, y int)和(x, y int)是等价的。

Multiple results

Named return values

Go的return如果是空的，但是有返回值，那么返回的是定义的同名的变量。

！这应只在短函数中使用，在长函数中使用会破坏可读性。

Variables

var定义variables，例如var c, python, java bool，是持久的。

Variables with initializers

var i, j int = 1, 2

Short variable declarations

在函数里，：=可以作为var + implicit type的替代品。

k := 3

Basic types

Zero values

  • 0 for numeric types,
  • false for the boolean type, and
  • "" (the empty string) for strings.

Type conversions

T(v) 转化 v 为 T 类型。

Go 只支持显示转化。
